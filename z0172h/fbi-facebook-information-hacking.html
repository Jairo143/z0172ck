¿Alguna vez quisiste hackear tu propia cuenta de Facebook? Pues este script en Python escrito por  xHak9x es lo más cercano a ello.

FBI (Facebook Information) es un herramienta de Information gathering precisa de una cuenta de Facebook.  Toda la información confidencial se puede recopilar fácilmente a pesar de que el objetivo/víctima convierta toda su privacidad a (solo para mí), información confidencial sobre residencia, fecha de nacimiento, ocupación, número de teléfono y dirección de correo electrónico.

No te engañaré, no vas a hackear facebook con esta herramienta!, PERO puedes obtener información confidencial de tus contactos (amigos) y incluso usar un bot para dar likes o comentarios en automático…. ¿te interesa?…

1. Instalación de fbi

El siguiente script escrito en Python solo requierer de ello (Python 2 exactamente), así que puedes utilizarlo en cualquier sistema operativo. Para este ejemplo, usaré mi Kali Linux.

Nos adueñamos de la herramienta ejecutando los siguientes comandos (incluye la instalación de requerimientos): 

                
                    apt update && apt upgrade
                    apt install git python2
                    git clone https://github.com/xHak9x/fbi.git
                    cd fbi
                    pip2 install -r requirements.txt

2. Ejecución y ayuda de comandos

Para iniciar la herramienta, ejecutamos el siguiente comando:

                
                    python2 fbi.py

Interfaz de la herramienta fbi.py

La descripción de FBI [Facebook Informations] en GitHub no ayuda mucho, así que ejecutemos el comando help para ver todas las opciones posibles…

3. ‘Hackeando’ perfiles de Facebook

Vale, antes que nada debemos iniciar sesión en la herramienta (a través del token). Colocamos el usuario y la contraseña y luego ejecutamos get_data para tener todos los datos de los perfiles amigos a nuestra disposición. 

                token
                <escribe_tu_usuario>
                <escribe_tu_contraseña>
                get_data

Con el permiso concedido de un fan de nuestra página de Facebook , y con el módulo get_info, he podido conseguir la siguiente información de perfil: 

                get_info
                <escribe_ID_perfil_amigo>

4. Bots de auto reacciones y auto comentarios en Facebook

Las maravillas de esta herramienta aumentan con su menú de Bot. FBI puede conseguir que tu cuenta se convierta en un bot de dar reacciones y/o comentarios a publicaciones de tus amigos.

Además, puedes aceptar todas las solicitudes pendientes de golpe, detener el following a todas las cuentas que sigues e incluso dejar tu perfil sin ninguna publicación!
