Fsociety es un paquete de herramientas de hacking: un Framework de pruebas de penetracion. Contiene todos los scripts que necesita un hacker. Fsociety contiene todas las herramientas utilizadas en la serie Mr. Robot.

Que es Fsociety


Si eres un apasionado de la ciberseguridad, debes haber oido hablar de la famosa serie Mr Robot. Fsociety es un Framework que se ha utilizado en la serie para llevar a cabo los diversos ataques de hacking. No quiero darte ningun spoiler, pero para aquellos que no han visto este programa de television, lo recomiendo encarecidamente.

Ventajas del Framework Fsociety:

  • Viene con un conjunto completo de herramientas para todas las etapas de prueba de penetracion.
  • Se puede usar en plataformas como Windows, Linux y Android.
  • Es super facil comenzar.
  • Esta automatizado, por lo tanto, te da un momento para tomar una taza de te.

Instalar fsociety


Para instalarlo en cualquier distribucion Linux (incluso Termux), ejecuta los siguiente comandos: 

                
                    git clone https://github.com/Manisso/fsociety.git
                    bash install.sh
                    python fsociety.py













En la caso utilices Docker: 

                
                    docker-compose build
                    docker-compose up -d
                    docker-compose exec fsociety fsociety
                    docker-compose down # destroys instance
