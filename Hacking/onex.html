onex: Instalador de Herramientas Hacking


Z0172CK

onex es un instalador de herramientas de hackeo y administrador de paquetes para hackers. Onex es una biblioteca de todas las herramientas Hacking para Termux y otras distribuciones de Linux. onex puede instalar cualquier herramienta de terceros o cualquier herramienta de hacking para ti.

Sobre onex


onex gestiona una gran cantidad de herramientas hacking que se pueden instalar con un solo clic. Con onex, puedes instalar todas las herramientas de hacking en Termux y otras distribuciones basadas en Linux. onex puede instalar mas de 370+ herramientas de hacking de Kali Linux.


Requisitos del sistema operativo


onex funciona en cualquiera de los siguientes sistemas operativos:


  • Android (Uso de la aplicación Termux)
  • Linux (sistemas basados ​​en Linux)

Como instalar onex


Abre la terminal y escribe los siguientes comandos. 

                
                    apt update
                    apt install git
                    git clone https://github.com/rajkumardusad/onex.git
                    chmod +x onex/install
                    ./onex/install



¿Como usar onex?



Modo CLI:

Para ayuda:

                
                    onex -h 
                    #O, onex help

Opciones:

  • onex install [tool_name] Instalar cualquier herramienta.
  • onex -i [tool_name ] Instalar cualquier herramienta.
  • onex search [tool_name] busca cualquier herramienta.
  • onex -s [tool_name] busca cualquier herramienta.
  • onex list Listar todas las herramientas.
  • onex list -a Listar todas las herramientas.
  • onex -l Listar todas las herramientas.
  • onex -l -a Listar todas las herramientas.
  • onex help consigue ayuda.
  • onex -h consigue ayuda.






Modo Menu


Escribe el siguiente comando para iniciar el modo de menu onex.

                
                    onex start

Ingresa un numero para una salida especifica:

  • (1) : para mostrar todas las herramientas disponibles y escribir el numero de una herramienta que deseas instalar.
  • (2) : para mostrar la categoria de herramientas.
  • (3) : si deseas actualizar onex.
  • (4) : si quieres saber sobre el creador.
  • (5) : para salir de la herramienta.
