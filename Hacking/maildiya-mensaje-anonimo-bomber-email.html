MailDiya: Remitente Anonimo y Bomber Emai


Z0172CK

MailDiya es un script para que permite utilizar un remitente anonimo de correo electronico y bombardear emails.

Una herramienta de codigo abierto para el remitente de correo electronico anonimo y el bombardeo de correo electronico.

  • El script requiere una conexion de red en funcionamiento para funcionar.
  • Mientras ejecuta este script, se debe usar un VPN.

Este guion es solo para fines educativos.


Caracteristicas:

  • Envio anonimo de correo electronico.
  • Bombardeo por correo electronico sin iniciar sesion.
  • Facil de usar.

Instalar MailDiya


Los metodos de instalacion de git no son universales y difieren entre las distribuciones, por lo tanto, la instalacion de git segun las instrucciones a continuacion puede no funcionar. Consulta como instalar git para tu distribucion de Linux. Los siguientes comandos proporcionan instrucciones para sistemas basados ​​en Debian.

Para usarlo en Termux, escribe los siguientes comandos: 

                
                    pkg install git
                    pkg install python2
                    git clone https://github.com/tuhin1729/MailDiya.git
                    cd MailDiya
                    pip2 install -r requirements.txt
                    python2 maildiya.py



Para distribuciones de Linux: 

                
                    git clone https://github.com/tuhin1729/MailDiya.git
                    cd MailDiya
                    pip install -r requirements.txt
                    python2 maildiya.py

En la opcion (1), el correo sera enviado dentro de 12 horas. Hasta el momento no se puede modificar el intervalo, pero es posible que llegue mucho antes.
