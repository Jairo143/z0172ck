¿Alguna vez quisiste hackear tu propia cuenta de Facebook? Pues este script en Python escrito por  xHak9x es lo mas cercano a ello.

FBI (Facebook Information) es un herramienta de Information gathering precisa de una cuenta de Facebook.  Toda la informacion confidencial se puede recopilar facilmente a pesar de que el objetivo/victima convierta toda su privacidad a (solo para mi), informacion confidencial sobre residencia, fecha de nacimiento, ocupacion, numero de telefono y direccion de correo electronico.

No te enganare, no vas a hackear facebook con esta herramienta!, PERO puedes obtener informacion confidencial de tus contactos (amigos) y incluso usar un bot para dar likes o comentarios en automatico…. ¿te interesa?…

1. Instalacion de fbi

El siguiente script escrito en Python solo requierer de ello (Python 2 exactamente), asi que puedes utilizarlo en cualquier sistema operativo. Para este ejemplo, usare mi Kali Linux.

Nos aduenamos de la herramienta ejecutando los siguientes comandos (incluye la instalacion de requerimientos): 

                
                    apt update && apt upgrade
                    apt install git python2
                    git clone https://github.com/xHak9x/fbi.git
                    cd fbi
                    pip2 install -r requirements.txt

2. Ejecucion y ayuda de comandos

Para iniciar la herramienta, ejecutamos el siguiente comando:

                
                    python2 fbi.py

Interfaz de la herramienta fbi.py

La descripcion de FBI [Facebook Informations] en GitHub no ayuda mucho, asi que ejecutemos el comando help para ver todas las opciones posibles…

3. ‘Hackeando’ perfiles de Facebook

Vale, antes que nada debemos iniciar sesion en la herramienta (a traves del token). Colocamos el usuario y la contrasena y luego ejecutamos get_data para tener todos los datos de los perfiles amigos a nuestra disposicion. 

                token
                <escribe_tu_usuario>
                <escribe_tu_contrasena>
                get_data

Con el permiso concedido de un fan de nuestra pagina de Facebook , y con el modulo get_info, he podido conseguir la siguiente informacion de perfil: 

                get_info
                <escribe_ID_perfil_amigo>

4. Bots de auto reacciones y auto comentarios en Facebook

Las maravillas de esta herramienta aumentan con su menu de Bot. FBI puede conseguir que tu cuenta se convierta en un bot de dar reacciones y/o comentarios a publicaciones de tus amigos.

Ademas, puedes aceptar todas las solicitudes pendientes de golpe, detener el following a todas las cuentas que sigues e incluso dejar tu perfil sin ninguna publicacion!
