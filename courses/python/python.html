La naturaleza del lenguaje


Desarrollado en 1990, Python es uno de los lenguajes de programacion de proposito general mas populares en los tiempos modernos.


El termino "proposito general" simplemente significa que Python se puede usar para una variedad de aplicaciones y no se enfoca en ningun aspecto de la programacion.


Un lenguaje de alto nivel


Python cae dentro de la categoria de lenguajes interpretados de alto nivel . Un lenguaje de alto nivel es uno que no puede ser entendido directamente por nuestra maquina. Hay un cierto grado de abstraccion en su sintaxis. Las maquinas generalmente estan diseñadas para leer codigo de maquina , pero la sintaxis de alto nivel no se puede convertir directamente a codigo de maquina.

Como resultado, primero debe convertirse a bytecode que luego se convierte a codigo de maquina antes de que se pueda ejecutar el programa.

Python es un lenguaje interpretado porque, durante la ejecucion, cada linea se interpreta en lenguaje de maquina sobre la marcha.

Sin embargo, si tomamos el ejemplo de C ++, el codigo debe compilarse primero en un ejecutable, y luego puede ejecutarse. En Python, podemos omitir este paso de compilacion (Python lo hace por nosotros detros de escena) y ejecutar directamente el codigo.


Legibilidad


Una de las principales razones del rapido crecimiento de Python es la simplicidad de su sintaxis. El idioma se lee casi como un simple ingles, lo que facilita la escritura de programas complejos.

Dado que no tiene mucha curva de aprendizaje, Python es un muy buen punto de entrada al mundo de la programacion para principiantes.

Historial de versiones


Python ha tenido varias actualizaciones importantes en el pasado. Python 2.7 fue ampliamente utilizado durante mucho tiempo, incluso despues del lanzamiento de nuevas versiones.

Sin embargo, Python 2.7 pronto sera desaprobado y reemplazado completamente por 3.xx versiones, conocidas como Python 3 . Las diferencias entre Python 2.7 y Python 3 son minimas, pero importantes.

Para mantenerse al dia con las últimas tecnologias, trataremos con Python 3 durante todo este curso.
