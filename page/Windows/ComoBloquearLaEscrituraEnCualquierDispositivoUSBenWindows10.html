Como bloquear la escritura en cualquier dispositivo USB en Windows 10

Incluso con todos los servicios en la nube disponibles, pocos renuncian al uso de una memoria USB. Es mas conveniente y mas simple que compartir y cargar en Internet, sin mencionar que no siempre hay Internet.

Estos pequenos gadgets ayudan a cualquier usuario al compartir archivos o simplemente llevarlos a otra computadora. Incluso con toda esta simplicidad, muchos pueden querer limitar su uso. Esto es lo que debes hacer cuando deseas bloquear una memoria USB contra la escritura en Windows 10.

Curiosamente, bloquear una memoria USB contra escritura en Windows 10 es muy simple. Todo esta en una zona de configuracion de este sistema y se puede activar en cualquier momento. Por lo tanto, es una solucion caracteristica que solo debe usar cuando tiene problemas y necesita ayuda.

Windows 10 te da el control


El primer paso de este proceso es simple y ciertamente conocido por la mayoria. Solo necesitas abrir el editor de registro de Windows 10 dando el comando regedit en el cuadro de busqueda. La opcion que aparece debe ser la que elija.



Tambien puedes presionar las teclas Windows + R, escribir regedit y presionar la tecla Intro.

Una vez dentro de esta aplicacion, debes navegar a la ruta:

Equipo\HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\StorageDevicePolicies

Aqui es donde cambiaras una clave para bloquear la escritura en cualquier unidad flash USB conectada.



Bloquear escritura en una memoria USB


Si esta clave no existe, deberian crear una nueva, llamada StorageDevicePolicies desde la zona de /Control (clic derecho, nueva clave). Esta es una clave normal y directa creada en Windows 10 regedit.




Debe haber una entrada llamada WriteProject. Se le debe asignar el valor 0 y se debe abrir con un doble clic. El valor mostrado debe cambiarse a 1. Luego, cierralo. De lo contrario, debes crear una nueva DWord (32 bits) con este nombre y el valor 1.
